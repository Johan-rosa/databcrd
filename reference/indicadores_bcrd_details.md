# Utility complement to handle the monetary indicators data

Utility complement to handle the monetary indicators data

## Usage

``` r
indicadores_bcrd_details
```

## Format

### `indicadores_bcrd_details`

A data frame with 39 rows and 6 columns:

- original_names:

  Names in the excel file

- labels:

  Names with meaning and without suffix

- short_names:

  unique names in snake case

- categoria:

  Category of the indicator

- nivel:

  level

- direct_parent:

  Direct parent
