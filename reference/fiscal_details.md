# Utility complement to handle the total imports data

Utility complement to handle the total imports data

## Usage

``` r
fiscal_details
```

## Format

### `fiscal_details`

A data frame with 102 rows and 6 columns:

- original_names:

  Names in the excel file

- labels:

  Names with meaning and without suffix

- short_names:

  unique names in snake case

- categoria:

  Category of the indicator

- nivel:

  level

- direct_parent:

  Direct parent
