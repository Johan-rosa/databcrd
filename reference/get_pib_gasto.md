# GDP by the expenditure approach, both real and nominal

GDP by the expenditure approach, both real and nominal

## Usage

``` r
get_pib_gasto(modalidad = "nominal", acumulado = FALSE, homogenea_91 = FALSE)
```

## Arguments

- modalidad:

  string indicating if "real" or "nominal"

- acumulado:

  logical, by default FALSE

- homogenea_91:

  logical, by default FALSE

## Value

a data frame
