# CPI by item data, from October 2020 to December 2025

CPI by item data, from October 2020 to December 2025

## Usage

``` r
data_ipc_articulos_2020_2025
```

## Format

### `data_ipc_articulos_2020_2025`

A data frame with 612 rows and 26 columns:

- id:

  Order in the original excel

- nombre:

  Item name, it can be a group, class or other level

- agregacion:

  This is the level of the given item

- group:

  Group name

- subgrupo:

  Subgroup name

- clase:

  Class name

- subclase:

  Subclass name

- articulo:

  Item name

- articulo:

  Item name

- date:

  Date

- year:

  Year

- mes:

  Month

- ponderacion_ipc:

  Weight of the element as proportion of the general basket

- indice:

  item index
